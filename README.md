# ansible-role-barotrauma

Ansible role to install and manage [Barotrauma](https://barotraumagame.com/wiki/Main_Page) using [LinuxGSM](https://linuxgsm.com/).

# Requirements

## Linux
- node.js

# Usage

```
- hosts: servers
  roles:
    - { role: frozenfoxx.barotrauma }
```
